Text2Event

March 25, 2026

Text2Event is a tiny productivity tool to create calendar events from text description.

It does not require any sign up and produces ICS files as well as deep links for Google Calendar, Outlook Live and Microsoft 365.

There are several quick access options like

  • Text2Event Chrome extension
  • Progressive Web Application (PWA) which in combination with Google Chrome on Android gives a native-like experience. I like share text with Text2Event PWA which automatically generates ICS file that is smoothlessly opened by Google Calendar.
  • and iOS Shortcut

The tool uses Claude API to parse text input.
